Song Kilns site (德文版本)

结构：
- index.html        ← 打开这个文件即可预览网页
- assets/ru.jpg     ← 本地图片（可替换为真实照片）
- assets/guan.jpg
- assets/ge.jpg
- assets/ding.jpg
- assets/jun.jpg

替换方法：
1) 把你的真实图片放到 assets/ 目录，保持同样文件名（或改名后同步修改 index.html 中的路径）。
2) 用浏览器打开 index.html 检查。
3) 部署到 GitHub Pages：将整个文件夹上传到公开仓库根目录，Settings → Pages → Deploy from a branch → main /(root) → Save。
